(Surfaces // Edit Curves / Detach Curves) ![curve01](img/maya/nurbs/curve/detach.png) - Joins NURBS curves together at their endpoints to form a new curve. You can select two or more curves for attaching. (Surfaces // Edit Curves / attach Curves) ![curve02](img/maya/nurbs/curve/attach.png) ## Cut Curve - Splits curves wherever they cross a cutting curve in the view. (Surfaces // Edit Curves / Cut Curve) ## Rebuild Curve - Performs various operations to modify the selected curve. (Surfaces // Edit Curves / Rebuild Curve) # Surface Creating ## Birail - Creates a surface by sweeping between a series of profile curves along two path curves. The resulting surface can maintain continuity with other surfaces. The Birail Tool has options for sweeping 1, 2, or 3 (or more) cross sectional curves along the two path (rail) curves. The resulting surface is interpolated through the profile curves. (Surfaces // Surfaces / Birail) ## Boundary - Creates a surface by filling between boundary curves. (Surfaces // Surfaces / Boundary) ## Bevel - Creates beveled transition surfaces from a profile curve. (Surfaces // Surfaces / Bevel) # Surface Editing ## Project Curve on Surface and Trim tool (Surfaces // Edit NURBS / Project Curve on Surface) ## Using the Sculpt Geometry tool (Surfaces // Edit NURBS / Sculpt Geometry Tool) ## Review - [Tumblr blog](http://2014ant3012.tumblr.com/) ## Tutorial / Inspiration - Maya tutorial videos for beginners: [webneel.com](http://webneel.com/maya-tutorials-videos-beginners) - More of Maya tutorials: [instantShift.com](http://www.instantshift.com/2009/03/06/45-incredible-maya-tutorials-around/) - NURBS modeling tutorials: [car rim](http://www.cgarena.com/freestuff/tutorials/maya/modeling-rim/modeling-rim.php), [car tutorial](https://www.tsplines.com/support/mayatutorials/24-mayatutorials/151-car-tutorial.html) ## Practice! ________________
