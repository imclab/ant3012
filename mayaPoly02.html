Working with symmetry - Using image planes to bring in reference sketches - Decomposing complex forms into simpler subcomponents, also known as block modeling - Using four-sided polygons - Maintaining polygons of relatively uniform size + numbers in pair and aspect ratio ### Toolsets: #### Start - Crete polygons **(Polygons // Mesh / Create Polygon Tool)** - Tip: Using grid to check general size and proportions **(Display / Grid)** - Using Extrude to expand polygon faces **(Polygons // Edit Mesh / Extrude)**, Tip: Check **Keep Faces Together**: this option ensures that edges do not separate as they are extruded. - Create eye balls **(Create / NURBS Primitives / Sphere)** - Tip: Using **Poly Count** and **Outliner** to check your Polygon data **(Display / Heads Up Display / Poly Count)** - Tip: **"g"** key for previous command #### Edit - Using component mode to move vertices and edges, tip: using selection tool **(Polygons // Select)** - Using Lattice deformer **(Animation // Create Deformers / Lattice)** - Using Sculpt tool **(Polygons // Mesh / Sculpt Geometry Tool)** #### Combine two meshes (eye and nose) - Combine multiple geometries to one **(Polygons // Mesh / Combine)** - Create faces between geometries **(Polygons // Edit Mesh / Append to Polygon Tool)** - Divide faces **(Polygons // Edit Mesh / Split Polygon Tool)** #### Create symmetry - Align vertices at the center: 1. choose all vertices at the center from top or front view. 2. Sanp to grid, tip: turn off Retain component spacing in the Move tool attribute editor. Check the image: ![move](img/maya/polygon01/movetool.png) - Mirror & Merge the geometry **(Polygons // Mesh / Mirror Geometry)** ![move](img/maya/polygon01/merge.png) #### Smoothing Polygon surfaces - To see a smooth version of the model, choose (Polygons // Normals / Soften Edge) - Or create Subdivision sufaces using Subdiv Proxy **(Polygons // Proxy / Subdiv Proxy)** #### Repeat edit surfaces ![crpo01](img/maya/polygon/eye01.png) ![crpo02](img/maya/polygon/eye02.png) ![crpo03](img/maya/polygon/eye03.png) ![crpo04](img/maya/polygon/eye05.png) ### Review - [Tumblr](http://ant3012.tumblr.com/) ### Inspiration - Low poly art: [Timothy J.
